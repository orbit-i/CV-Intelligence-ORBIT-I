# CV Intelligence & Offer Automation Platform

This project is being reshaped into a platform for:

- uploading candidate resumes
- extracting candidate details from documents
- classifying the domain of the candidate
- generating tailored job offers

## Project structure

- app.py: Flask application entrypoint
- config/: app settings and prompt templates
- ui/: UI blueprints and reusable view components
- parser/: resume parsing utilities
- ai/: domain classification and offer generation logic
- services/: business services for candidate and offer workflows
- models/: domain models
- uploads/: uploaded documents
- output/: generated outputs
